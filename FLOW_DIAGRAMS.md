### 2. Resume Upload & Processing Flow

```
START
  │
  ▼
┌─────────────────────┐
│ User selects file   │
│ or enters URL       │
└──────────┬──────────┘
           │
           ▼
    ┌──────────────┐
    │ File Upload? │
    └──┬────────┬──┘
       │ Yes    │ No (URL)
       ▼        ▼
┌──────────┐  ┌──────────┐
│ Multer   │  │ Download │
│ receives │  │ from URL │
│ file     │  │ via Axios│
└────┬─────┘  └────┬─────┘
     │             │
     └──────┬──────┘
            ▼
┌─────────────────────┐
│ Save to /uploads/   │
└──────────┬──────────┘
           ▼
┌─────────────────────┐
│ Detect file type    │
│ (PDF/DOCX/TXT)      │
└──────────┬──────────┘
           ▼
┌─────────────────────┐
│ Extract text using: │
│ - pdf-parse (PDF)   │
│ - mammoth (DOCX)    │
│ - fs.read (TXT)     │
└──────────┬──────────┘
           ▼
┌─────────────────────┐
│ Parse with OpenAI:  │
│ - Name, Email       │
│ - Skills, Experience│
│ - Education         │
└──────────┬──────────┘
           ▼
┌─────────────────────┐
│ Extract Profile     │
│ Links (GitHub,      │
│ LinkedIn, LeetCode) │
└──────────┬──────────┘
           ▼
┌─────────────────────┐
│ Create Resume       │
│ Object with:        │
│ - Parsed data       │
│ - Profile links     │
│ - Timestamp         │
└──────────┬──────────┘
           ▼
┌─────────────────────┐
│ Store in            │
│ resumeDatabase      │
└──────────┬──────────┘
           ▼
┌─────────────────────┐
│ Return success      │
│ response with       │
│ candidate name      │
└──────────┬──────────┘
           ▼
          END
```
